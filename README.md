# Github-Profile-Analyzer

It allow users to enter a GitHub username and return repositories and visualize commit activity.

## 🚀 Features
- ✨ Modern UI with ShadCN and Bootstrap
- 🔎 Username input to fetch GitHub data
- 
## 📦 Tech Stack
- [Vite]
- [React]
- [TypeScript]
- [ShadCN UI]
- [Bootstrap 5]
